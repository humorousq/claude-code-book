# 序言

## 关于本书

本书是《Claude Code 实战工作流指南》的完整版本，专注于帮助读者深度掌握 **Everything Claude Code 扩展包**的核心思想和使用方法。

## 为什么需要这本书？

Everything Claude Code 是一个生产就绪的扩展包，包含 50+ Skills、20+ Agents 和完整的自动化工作流。然而，许多用户在使用时面临以下挑战：

- ❌ 不理解扩展包的设计理念和核心价值
- ❌ 不知道如何选择合适的 Skills 和 Agents
- ❌ 难以将扩展包应用到实际项目中
- ❌ 团队协作时缺少统一的最佳实践

本书旨在解决这些问题，通过系统化的讲解，让你真正掌握 Everything Claude Code。

## 内容来源

本书基于 everything-claude-code 仓库的中文内容，按工作流重新组织和扩展，聚焦于扩展包的核心思想和实际应用。

## 如何阅读

### 学习路径

**初学者路径**：
1. 第一部分：理解扩展包的设计理念（必读）
2. 第二部分：按顺序学习核心工作流
3. 第三部分：根据需要跳转阅读

**有经验用户路径**：
1. 快速浏览第一部分，理解核心概念
2. 直接跳到感兴趣的工作流章节
3. 需要时查阅附录

### 章节依赖关系

```
第1章（设计理念）→ 第2章（安装配置）
    ↓
第3章（需求规划）→ 第4章（TDD）→ 第5章（代码审查）→ 第6章（CI/CD）
    ↓
第7-10章（高级应用，可按需跳转）
```

## 预期收获

通过本书，你将能够：

- ✅ 理解 Everything Claude Code 的设计理念和核心价值
- ✅ 掌握从需求到部署的完整工作流
