## E.16 术语使用约定

### 大小写规则

1. **专有名词**：首字母大写
   - Claude Code
   - TypeScript
   - Python
   - Docker

2. **缩写**：全大写
   - API
   - LLM
   - MCP
   - TDD

3. **命令**：小写
   - `/tdd`
   - `/plan`
   - `/code-review`

4. **工具名**：首字母大写
   - Read
   - Write
   - Bash
   - Grep

### 翻译原则

1. **通用术语**：保持英文或使用广泛接受的中文翻译
   - API → API
   - Git → Git
   - Docker → Docker

2. **概念术语**：提供中英文对照
   - Agent → 代理
   - Skill → 技能
   - Hook → 钩子

3. **技术术语**：优先使用英文
   - TypeScript（非"类型脚本"）
   - JavaScript（非"爪哇脚本"）
   - React（非"反应"）
