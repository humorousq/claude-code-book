## 🚀 快速开始

### 本地开发

```bash
# 克隆仓库
git clone https://github.com/username/claude-code-book.git
cd claude-code-book

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 http://localhost:5173

### 构建

```bash
# 构建网站
npm run build

# 预览构建结果
npm run preview

# 生成 PDF
npm run pdf
```

### 工具脚本

```bash
# 修复章节编号
npm run fix

# 验证构建
npm run verify

# 运行测试
npm test
```
